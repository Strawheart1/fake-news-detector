# Fake News Detector 📰🔍

A Python-based fake news detector that analyzes text 
and returns a confidence score on whether it is real or fake.

## How It Works
- User inputs a headline or article text
- The model analyzes the text
- Returns a confidence score of how likely it is fake or real

## Features
- Confidence score output
- Works best on full sentences with context

## Tools Used
- Python 3.x
- VS Code

## ⚠️ Disclaimer
This tool is not 100% accurate. It works best on full sentences 
with context. Always verify with trusted fact-checking sources.
